# ohos-usageStatsQuery

Application usage statistics query tool for OpenHarmony system.

## Description

This CLI tool provides comprehensive application usage statistics query capabilities, including:
- Application idle status
- Application usage period
- Usage statistics by time interval
- Application event records
- Application group information
- High-frequency usage applications
- Module usage records
- Notification event statistics
- High-frequency usage periods
- Today's latest usage time

## Installation

This tool is installed as part of the system image at:
- Path: `/system/bin/cli_tool/executable/ohos-usageStatsQuery`
